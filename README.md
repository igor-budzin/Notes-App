Notes App
===================
Link for preview (only client side, without db and api): [link][1]

### Technologies
**Front-end:** 
:   React.js
:   Redux
:   Axios

**Front-end:** 
:  Node.js (Express)
:  Mongodb

### Run
Install dependencies
```
npm install
```
start local dev server
```
npm run server
```
and open link in browser
```
http://localhost:8080
```
